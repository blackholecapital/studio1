## Recommendations (for future steps, NOT this baseline)

1. The 16 orphaned source files appear to be **deprecated component versions** that were superseded by inline implementations in App.tsx and MobileApp.tsx. They may contain useful logic to extract.
2. `store.ts`, `layoutPresets.ts`, and `defaultContent.ts` contain initialization logic that is **duplicated inline** in App.tsx. Consolidation would be beneficial.
3. The committed `dist/` assets should be moved to `public/` or served from R2 rather than checked into git.
4. `types.ts` defines `TabKey` with "scraper" and "prototype" values, but the corresponding panels are orphaned. The type is still valid as a union discriminator.
